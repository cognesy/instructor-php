The following list shows the supported providers:

- A21
- Anthropic
- Azure
- Cerebras
- Cohere
- Cohere (OpenAI compatible API)
- DeepSeek
- Fireworks
- Gemini
- Gemini (OpenAI compatible API)
- Groq
- MiniMaxi
- Mistral
- Moonshot
- Ollama
- OpenAI
- OpenRouter
- Perplexity
- SambaNova
- Together
- XAI

Check [the cookbook](/cookbook/introduction) for examples of how to use them.
