# Symfony Quickstart

The Symfony package is currently in the package-foundation phase.

What exists now:

- the Composer package definition
- the namespace root `Cognesy\Instructor\Symfony\`
- the package documentation surface

What lands next:

1. the bundle and extension entrypoints
2. the public `instructor` config tree
3. container bindings for Instructor and Polyglot runtime services

Until those tasks land, use the planning documents under `research/2026-03-19-symfony-batteries-included-plan/` as the source of truth for the intended package shape.
