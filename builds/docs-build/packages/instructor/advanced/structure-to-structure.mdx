---
title: 'Structure To Structure'
description: 'Start from an array or object instead of raw text.'
---

`withInput(...)` converts arrays and objects into messages before execution.

```php
$result = (new StructuredOutput)
    ->withInput(['name' => 'Jane', 'bio' => 'Engineer from Berlin'])
    ->withResponseClass(Profile::class)
    ->get();
// @doctest id="a1b4"
```

Use it when your source data is already structured but you still want a model-driven transformation.
