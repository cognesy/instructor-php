---
title: Scalars
description: 'Extract one value without creating a full class.'
---

Use `Scalar` when you want one typed value.

```php
use Cognesy\Instructor\Extras\Scalar\Scalar;
use Cognesy\Instructor\StructuredOutput;

$age = (new StructuredOutput)
    ->with(messages: 'Jason is 28 years old.', responseModel: Scalar::integer('age'))
    ->get();
// @doctest id="3a3a"
```

Helpers available on `Scalar`:

- `string(...)`
- `integer(...)`
- `float(...)`
- `boolean(...)`
- `enum(...)`

If you already have a pending request, the type helpers on `StructuredOutput` and `PendingStructuredOutput` can also be useful, such as `getString()` or `getInt()`.
