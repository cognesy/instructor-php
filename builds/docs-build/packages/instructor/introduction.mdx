`cognesy/instructor-struct` is the core structured-output package in InstructorPHP.
It gives you a small API for asking an LLM for data that fits a schema you control.

The package is centered on four types:

- `StructuredOutput` builds the request
- `StructuredOutputRuntime` configures provider and runtime behavior
- `PendingStructuredOutput` executes lazily
- `StructuredOutputStream` exposes streaming updates

## The Core Flow

1. Define a response model.
2. Build a request.
3. Read the result.

```php
use Cognesy\Instructor\StructuredOutput;

final class City {
    public string $name;
    public string $country;
}

$city = StructuredOutput::using('openai')
    ->with(
        messages: 'What is the capital of France?',
        responseModel: City::class,
    )
    ->get();
// @doctest id="529d"
```

## What This Package Covers

- Typed objects, schema arrays, and helper response models
- Validation and retry-aware extraction
- Streaming partials and sequences
- Runtime configuration through `LLMConfig` and `StructuredOutputConfig`
- Events and pipeline extension points
