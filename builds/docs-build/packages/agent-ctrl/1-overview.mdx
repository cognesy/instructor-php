---
title: Overview
description: 'Unified API for Claude Code, Codex, and OpenCode CLIs.'
---

`agent-ctrl` gives you one fluent API (`AgentCtrl`) to run different code-agent CLIs.

Supported bridges:

- Claude Code
- OpenAI Codex
- OpenCode

All of them return the same response type: `AgentResponse`.
